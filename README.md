# MediScan AI

**AI-Powered Medical Image Analysis Platform**

HealTech Innovators Hackathon 2025 - AI for Healthcare Track

## The Problem

- 2.6 billion people lack access to basic healthcare
- 4-8 weeks average wait for specialist consultation
- $150-300 cost per dermatology visit
- 40% survival rate drop when diseases detected late

## Our Solution

MediScan AI provides instant, free AI-powered medical image analysis accessible to anyone with a smartphone.

## Key Features

- **Skin Lesion Detection**: Melanoma, basal cell carcinoma, benign moles
- **Chest X-Ray Analysis**: Pneumonia detection
- **Real-time Analysis**: Results in under 3 seconds
- **Mobile-First Design**: Works on any device
- **Privacy-First**: No data storage, anonymous usage
- **Free Forever**: No subscriptions or hidden costs

## Tech Stack

- React 18 + TypeScript + Tailwind CSS
- Vite for build tooling
- Lucide React for icons
- Responsive design

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Deployment to Vercel

1. Push this repo to GitHub
2. Import to Vercel
3. Deploy (auto-detected as Vite project)

No configuration needed!
